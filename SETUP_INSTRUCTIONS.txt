STEP 1: Navigate to Dashboard App Directory
──────────────────────────────────────────────────────────────────────────────

STEP 2: Install Dependencies
──────────────────────────────────────────────────────────────────────────────
  $ npm install
  
  This will download and install:
  • React & React-DOM (UI framework)

STEP 3: Start the Dashboard
──────────────────────────────────────────────────────────────────────────────
  $ npm start
  
  Your dashboard will be ready at: http://localhost:3001
  
  Open your browser and paste that URL! 🎉

🛠️  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

❌ "Port 3001 already in use"
   → Kill the existing process:
     $ lsof -ti:3001 | xargs kill -9
     $ npm start

❌ "Cannot find module"
   → Reinstall dependencies:
     $ rm -rf node_modules package-lock.json
     $ npm install
     $ npm start

❌ "npm command not found"
   → Node.js is not installed. Download from https://nodejs.org

❌ Dashboard looks wrong/doesn't load
   → Clear browser cache (Ctrl+F5 or Cmd+Shift+R)
   → Check browser console for errors (F12)
   → Ensure dev server is running in terminal

❌ Changes to code not showing
   → The dev server auto-reloads. Wait a few seconds.
   → If it doesn't work, stop (Ctrl+C) and restart: npm start
